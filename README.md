# Node.js Weather API

> Backend API for the Weather application to provide weather details of given location

## Usage

It convert address into geocode and then give weather details. It uses two APIs :-
- openweathermap
- mapbox

## Install Dependencies

```
npm install
```

## Run App

```
node index.js
```
